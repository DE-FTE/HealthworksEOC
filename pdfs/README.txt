DROP YOUR PDF FILES IN THIS FOLDER
===================================

Any .pdf file placed here will automatically appear in the
HealthworksAI sidebar when you run the app.

Click a PDF name in the sidebar to index it and start chatting.

For Vercel production deployment, use Vercel Blob instead:
  1. Set PDF_SOURCE=vercel-blob in your environment variables
  2. Upload PDFs via: npx vercel blob put <name.pdf> <path/to/file.pdf>
  3. Or use the Vercel dashboard → Storage → Blob → Upload
